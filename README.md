# esp32 mini simulations
Some little simulations using ESP32 DEVKIT V1 and SH110X LCD

- Dasai Muchi AI eyes
- Bubbles
- Box Jump (Dino Jump)
- Fish Pond
- Fire Place
- Starfield
- Asteroid Game
- Pipes
- Sinewave
- Sort of 3d tunnel

Check the **photos** folder to see the actual image from the LCD
  
Here's the sample demo using Wokwi.com - The code by the way supports both SSD1306 and SH110X OLEDS
